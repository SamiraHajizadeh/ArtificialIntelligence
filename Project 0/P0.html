The cons and pros of replacing NaNs with the mean value +: -the mean value remains the same -it's a better repressentative of the feature than a random number (e.g. 0) -: -it is heavily influenced by the outlier data example: mean([0, 1, 2, 1000]) = 250.75 which is not a good repressentative -although it can be a repressentative, it's the worst one (especially if the data is skewed) NOTE: Ideally, we want to replace NaNs with values in a way that the central tendency of data remains the same after replacement.

Choosing a feature for making predictions In order to predict properly, we should choose a feature that shows the most difference between the two target situations (y==yes and y==no). As seen above, the least overlap between the two curves belongs to the 'duration' feature. In addition, this is the only plot in which the peaks of the curves are on different sides of the intersection. We can explain this by assuming that a customer who is interested in a call from the bank, is probably more eager to learn other advertisements as well, and is more likely to be convinced to make a long term deposition. ***NOTE: if the peaks of the curves are on the same side of the intersection point, since their highest possibility of occurrence remains on the same side, using the intersection point will be most probably in vain. Therefore, we conclude that our feature should be duration and the intersection point used must be 0.525252525252526
